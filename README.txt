for using this module note that you have enabled low sequrity in gmail.

In handMail you have to type the content but in intelligentMail you have to speak the content and
it will converted into string and will be mailed.

How to use handMail - 
    import sendEmail as se
    se.handMail(from_='someone@gmail.com', pass_='********', to_='some_one@gmail.com', content='Hello')
    #type your content

How to use intelligentMail -
    import sendEmail as se
    se.intelligentMail(from_='someone@gmail.com', to_='some_one@gmail.com', pass_='********')
    #speak your content which you want to send.
